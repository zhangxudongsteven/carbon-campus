管理建议与预警

超限预警

2022-2023年单位面积碳排放超推荐限值，建议优先核查科研楼（占比36.12%）、宿舍（占比32.05%）等高能耗功能区用能情况。

趋势分析

2023年同比下降1.03%，显示管控措施有效。建议继续推进节能改造，争取2024年降至80kg/㎡以下。

达标路径建议

  • • 短期目标（2024年）：通过照明LED改造、空调系统优化，降至78kg/㎡·年
  • • 中期目标（2025年）：推进屋顶光伏、地源热泵试点，降至75kg/㎡·年
  • • 长期目标（2030年）：建成绿色校园示范区，稳定在70kg/㎡·年以下
